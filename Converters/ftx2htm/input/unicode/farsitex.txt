#	General notes:
#
#	This table contains the data on how IRAN SYSTEM characters
#       map into Unicode.
#
#	Format:  Three tab-separated columns
#		 Column #1 is the IRAN SYSTEM code (in hex as 0xXX)
#		 Column #2 is the Unicode (in hex as 0xXXXX),
#			plus additional formatting codes
#		 Column #3 the near-Unicode name
#			(follows a comment sign, '#')
#
#	The entries are in IRAN SYSTEM order
#
#	Any comments or problems, contact <roozbeh@sina.sharif.ac.ir>
#
#	Notes:
#
#	1. Note that IRAN SYSTEM is not a standard, it is almost a corporate
#	character set that found it way through the Iranian user community.
#	So, there is no standardization paper to refer to. The lower half
#	of character space is the same as ASCII.
#
#	2. The characters in the IRAN SYSTEM standard are saved, and
#	transmitted in a visual order. So converting Persian/Latin text
#	coded in IRAN SYSTEM to Unicode with bidirectional behaviour may be
#	hard, and may depend on application. Because of this, converting
#	from any other character set to IRAN SYSTEM is not recommended,
#	because of the loss of data.
#
#	3. There may be two, three, or four codes per character in most cases:
#	for initial, medial, isolated, and final forms of the letters, some of
#	which are unified in most of the cases.	I have added two tags before
#	and after the letters: <ZWJ> and <ZWNJ>.
#	
#       Their use after the code means that a ZERO WIDTH JOINER (0x200D) or a
#	ZERO WIDTH NON JOINER (0x200C) should be added after the character if
#	the character is not followed by a right-join causing character, or
#       a non-joining character respectively (in the manner of The Unicode
#	Standard Version 2.0, pages 6-24 and 6-25). The same can be said when
#	the code is preceded by the tag.
#
#	In informal words, they should be added if and only if there is a need.
#	For example, Persian word "khaane-haa" (houses) that is coded in
#	IRAN SYSTEM as:
#		0xA1 0x91 0xF7 0xF9 0xFA 0x91,
#	is equal to the Unicode character sequence:
#		0x062E 0x0627 0x0646 0x0647 0x200C 0x0647 0x0627.
#	
#	4. The near-Unicode names are only informative. In most of the cases,
#	there isn't exactly one Unicode character per IRAN SYSTEM character.
#
