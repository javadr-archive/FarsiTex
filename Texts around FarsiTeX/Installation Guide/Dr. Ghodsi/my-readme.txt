
How to get FarsiTeX:

1- Go to http://farsitex.org -> download
2- Download
    
     farsitex-1.0-alpha-3.zip
     miktex-small-2.0.zip
     farsitex-editor-alpha9.1.exe

3- First file contains readme.txt. Follow the installation steps
   exactly.

or get the files from
http://ce.sharif.ac.ir/~ghodsi/farsitex-samples/FarsiTeX%20download/
that contains readme.txt

4- Download the Persian manual from 
   
   http://ce.sharif.edu/~ghodsi/farsitex-samples/My%20Sample%20Files/farsitex-manual/man.ps

5- For professional use, you need to get a copy of LaTeX manual.

6- For drawing nice pictures, use LaTeXcad.

7- Use [farsi,lotusfont] (instead of just [farsi]) to typeset everything in
lotusfont.
