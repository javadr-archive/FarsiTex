That is because MikTeX 2.5 does not have e-Tex extended mode, it has
pdfetexe instead.
If you are trying to run FarsiTeX on MikTeX 2.5, you are required to do
the following steps:
1-Let's assume you have installed MikTeX2.5 on your computer.
2- download the file FarsiTeX for windows from www.farsitex.org and then
extract it to wherever you like.
3- copy this in your FarsiTeX extraction to MikTeX files:
FarsiTeX\MikTeX 2.5\miktex\bin------>MikTeX 2.5\miktex\bin
4- Go to the drive where you have installed MiKTeX 2.5, open MiKTeX 2.5,
open fonts, open enc, open dvips and finally create a new folder and name
it farsitex.
5- copy this in your FarsiTeX extraction to MiKTeX files:
FarsiTeX\MikTeX 2.5\fonts\enc\dvips\farsitex---->MikTeX
2.5\fonts\enc\dvips\farsitex

Note: you either need to do do step 6 or 7.
6- If you do not have windows vista, click on start, click on run and then
type mo.
7- If you have windows vista, go to the drive where you have installed
MikTeX 2.5, open the folder, open miktex, open bin and then double click on
mo.
8- click on general tab and then click on Refresh FNDB to update your
MiKTeX.
9-click on Formats, click on new and fill in the blank spaces as follow:
Format Name: FarsiTeX
Compiler: pdfetex
Input File Name: farsitex.ini
Output File Name: farsitex.efmt
Preloaded Format: 
Description: FarsiTeX
10-click ok, and again click ok and then click on build and then click on
ok.
11- open DOS and give the following command:
initexmf --mkmaps
12- install the FarsiTex fonts in your system fonts.
