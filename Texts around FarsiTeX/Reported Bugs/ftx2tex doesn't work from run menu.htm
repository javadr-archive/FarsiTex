Dear friends,
I had the same problem, I mean in Windows systems it seems 
that no command of the Run Menu is working properly inside 
the editor. I solved it by a trick and you can do that 
too. Here is what I have done (I presume that your 
localtxmf tree is c:\localtexmf):
1. Go to c:\localtexmf\miktex\bin and open goftex.bat with 
notepad editor
2. You can see the following lines in this file:

@echo off
%3
cd %1
ftx2tex %2
etex "&farsitex" %2

Change and save it as follows:

@echo off
%3
cd %1
ftx2tex %2.ftx
etex "&farsitex" %2.tex

Now farsitex can be run from the editor.
For ftx2tex change and save convert.bat as follows:

@echo off
%3
cd %1
ftx2tex %2.ftx
 
good luck
