This folder contains the MiKTeX 2.0 distribution. You may want to visit the project page (www.MiKTeX.org), for more information about MiKTeX.

The distribution consists of the MiKTeX Setup Wizard (setupwiz.exe) and several Zip files.

Download Instructions:

  1. Download the MiKTeX Setup Wizard.
  2. Download the Zip files (see table below) that you want to install. The MiKTeX distribution is composed of five levels:

    Level 1Basic applications.
    Level 2Adds more advanced applications.
    Level 3For the experienced TeX user.
    Level 4Adds esoteric applications.
    Level 5Includes source code for the the LaTeX2e packages. It's not normally necessary to download this.

    You should download at least all level 1 Zip files.

Levels above 100 include language dependent packages.
